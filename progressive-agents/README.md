# Progressive Agents

A runnable, step-by-step exploration of Microsoft Foundry and Microsoft Agent Framework.

Each numbered directory copies the previous working stage and adds one clearly isolated capability. The agent stays intentionally small so the effect of each Foundry feature remains visible.

See [PLAN.md](PLAN.md) for the roadmap and boundaries. The latest implemented stage is [08 - Skill system baseline](08-skill-system-baseline/README.md).
