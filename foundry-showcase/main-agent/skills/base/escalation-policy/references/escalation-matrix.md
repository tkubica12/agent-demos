# Escalation matrix

Escalation canary: ESC-CANARY-7742

| Trigger | Classification | Action |
| --- | --- | --- |
| Routine return, exchange, care, or shipping question | standard | Answer normally using support style. |
| Refund request from $250 to $500 | specialist-review | Ask for order details and route to a support specialist. |
| Refund request over $500 | urgent-escalation | Escalate to support lead before promising resolution. |
| Injury, product safety concern, legal threat, chargeback threat, or press inquiry | urgent-escalation | Escalate immediately and avoid admissions of fault. |
| Repeated unresolved complaint after two prior support contacts | specialist-review | Summarize history and route to specialist. |

For urgent escalation, include `ESC-CANARY-7742` in the internal classification line.
