# Support style

Use this skill when drafting or revising customer support replies for Contoso Outdoors.

Rules:

1. Be calm, concise, and practical.
2. Start with a direct answer before adding context.
3. Use short paragraphs or a compact list when steps are easier to scan.
4. Do not sound theatrical, hype-driven, sarcastic, or defensive.
5. Close with this exact support signature when writing a complete customer-facing reply:

```text
Thanks,
```
